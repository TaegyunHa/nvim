# nvim

## Set config path

Set environment variable:
- Variable: `XDG_CONFIG_HOME`
- Value: `%USERPROFILE%\.config`

#### Windows PowerShell

```pwsh
[Environment]::SetEnvironmentVariable("XDG_CONFIG_HOME", "$env:USERPROFILE\.config", "User")
```
